MQTT Protocol
Publish-Subscribe Model

📤 Publisher
Mobile / Web App
Sends commands to MQTT broker by publishing to a specific topic.
  • Publishes to topics
  • Doesn't know subscribers
  • Fire-and-forget model
  • QoS levels 0, 1, 2
PUBLISH
Topic: home1/light
Payload: ON
home1/light
→
🔀 HiveMQ Broker
Cloud MQTT Broker
Routes messages from publishers to all matching subscribers efficiently.
● home1/light → ESP32-A
● home1/fan → ESP32-B
● home2/light → ESP32-C
● home2/ac → ESP32-D
broker.hivemq.com:1883
home1/light
→
📥 Subscriber
ESP32 + Relay
ESP32 subscribes to specific topics and triggers relays on incoming messages.
  • Subscribes to topics
  • Receives matching messages
  • Triggers GPIO pins
  • Controls relay state

Core Components

🔲
ESP32

ESP32

// MICROCONTROLLER
Built-in WiFi + Bluetooth — No external modules needed for wireless communication
Dual-Core 240 MHz — Xtensa LX6 processor for parallel task execution
Low Cost ~₹300 — Affordable for IoT deployments at scale
GPIO Pins — 34 programmable pins for relay control and sensor integration
MQTT Compatible — PubSubClient library enables easy MQTT messaging

Relay Module

// HIGH VOLTAGE INTERFACE
Electrical Isolation — Optocoupler separates 3.3V logic from 230V AC
Safe AC Control — Controls fans, lights, AC units safely via GPIO signal
10A Load Capacity — Handles high-current appliances without damage
Active LOW Trigger — GPIO LOW from ESP32 activates relay coil

RELAY 4CH

HiveMQ Broker

// MESSAGE BROKER
Topic-Based Routing — Routes messages to exact subscribing devices
Device Clustering — Manage multiple ESP32 clusters under one broker

Key Advantages

🌍
Remote Access
Control your home appliances from anywhere in the world using just an internet connection.
📡
Low Bandwidth
MQTT protocol uses minimal data — approximately 2KB overhead — ideal for IoT with limited bandwidth.
🏗️
Scalable Architecture
Single HiveMQ broker handles thousands of devices across multiple locations simultaneously.
💰
Cost Effective
ESP32 + relay module costs under ₹500. HiveMQ public broker is free. Minimal infrastructure needed.
🔄
Converts Traditional
No need to replace existing switchboards. ESP32 + relay simply interfaces with existing wiring.
